카카오맵 앱키 설정이 필요합니다

  1. developers.kakao.com 에서 앱 생성 후 JavaScript 키 발급
  2. 내 애플리케이션 → 플랫폼 → Web에 서빙 도메인 등록 (예: http://localhost:8000)
  3. 이 파일 상단 const KAKAO_KEY = "..." 에 발급받은 키 입력
  4. 폴더에서 python -m http.server 8000 실행 후 http://localhost:8000 접속

※ 카카오맵은 보안상 file:// 직접 열기로는 동작하지 않습니다.
